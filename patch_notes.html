    • Item tracker has option to only show quests that you can accept now.

    • Added optional scroll bar to item tracker.

    • New client patch: v43
    • There is now a new feature that lets you track attunable items in your current zone. It's disabled by default but can be enabled in Interface -> Addons -> Item Tracker.
    • Attunable items list now lets dual class characters select to filter for one of the classes.
    • Maybe fixed bug where some dual class combinations could not parry despite having the parry skill.
